Quick Guide:
1 beat per note by default.
Notes:
0-127
midi values
-
rest
x
pitch generated by magenta.js MusicRNN model
(may take a few seconds)
Rhythms:
( )
all notes inside fit into one beat
n *
repeat following note or (...) n times, no change to duration
n **
repeat following note or (...) n times, fit all repeats into original duration
Start each loop with a key followed by an equals sign.
Key cannot contain spaces.
Begin each loop on a new line.
Don't worry about extra spaces between notes/symbols.
Here's an example to get you started:
a = 2*60 3**63 (65 65) x
